## 가로 스크롤 메뉴 & 인피니티 스크롤

> 가로 스크롤 카테고리 메뉴 및 해당 카테고리 상품에 대한 인피니티 스크리롤 구현.

### 요구사항

- SPA(Single Page Application)로 개발합니다.
- 프론트엔드 구현 방법은 제한 없습니다. (Angular, React, Vue, jQuery..)
- 접기, 펼치기 기능을 통해 모든 메뉴를 한번에 볼 수 있습니다.
- 클릭 시 현재 메뉴를 활성화 하고 가운데로 이동 되어야 합니다.
- 아래로 스크롤 시 메뉴가 사라지지 않고 상단에 고정 되어야 합니다.

### 인피니티 스크롤

### 요구사항

- 현재 활성화된 카테고리 상품을 인피니티 스크롤을 통해 50개씩 추가합니다.
- 각 카테고리 상품은 nCurPage nEndPage
- 메뉴 클릭 시 인피니티 스크롤 초기화후 1페이지 부터 시작됩니다.
- 현재 활성화된 메뉴를 클릭 시 아무런 작동이 되지 않아야 합니다.
- 상품 클릭 페이지 이동 시 현재 데이터 기억
- 뒤로가기(히스토리백) 시 이전 데이터 상태 유지
